Detecting_Weibo_Fake_Accounts
=============================

识别微博营销号的机器学习模型

iJoin社会创新咨询Innovation Center微博研究项目

合作者：陶鑫

1) SVM模型

在R中使用SVM模型，用于识别营销号。

算法：R package e1071 default算法

模型参数：degree = 3，cost = 1，kernel ="radial"，type = "class"

试验次数：500次

具体情况参见：http://markfu1022.weebly.com/1/post/2014/02/svm.html
